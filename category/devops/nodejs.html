Node.js安装与配置

maiaimei 2021/10/9 Node.jsnpm

下载地址：

  • https://nodejs.org/en/ (opens new window)

配置环境变量：

  • NODE_HOME = Node.js的安装目录
  • Path 添加 %NODE_HOME%;%NODE_HOME%\npm_global
# 查看 nodejs 版本
node -v

# 查看 npm 版本
npm -v

# 查看所有 config 变量
npm config list
# 配置全局目录
npm config set prefix "D:\\nodejs\\npm_global"
# 配置缓存目录
npm config set cache "D:\\nodejs\\npm_cache"
# 配置淘宝镜像
npm config set registry "https://registry.npmmirror.com"

# https://npmmirror.com/
# 安装 cnpm
npm install -g cnpm --registry=https://registry.npmmirror.com

# 安装运行时依赖
npm i -S <packname>
npm i --save <packname>

# 安装开发时依赖
npm i -D <packname>
npm i --save-dev <packname>

# 全局安装依赖
npm i -g <packname>
npm i --global <packname>

# 清缓存
npm cache clean -f

maiaimei 2021/10/17

# 修改终端

按住Crtl+Shift+P，在弹出的窗体中输入Preferences:Open User Settings，找到

在 setting.json 中编辑

{
    "terminal.integrated.profiles.windows": {
        "bash": {
            "path": "D:\\Git\\bin\\bash.exe",
            "args": []
        }
    },
    "terminal.integrated.defaultProfile.windows": "bash"
}
